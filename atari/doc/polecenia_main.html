Polecenia ATARI

Strona Główna Polecenia ATARI Błędy interpretera ATARI Błędy programu
FD BK RT LT PU PD HT ST CS REPEAT TELL ASK

FD

Użycie: FD wartość

Przesuwa żółwia o podaną wartość do przodu. Jeśli żółw(ie) zostawiają ślad zostanie narysowana linia na planszy.
Uwaga: wartość musi być liczbą całkowitą.

BK

Użycie: BK wartość

Przesuwa żółwia o podaną wartość do tyłu. Jeśli żółw(ie) zostawiają ślad zostanie narysowana linia na planszy.
Uwaga: wartość musi być liczbą całkowitą.

RT

Użycie: RT wartość

Obraca żółwia(ie) o podaną liczbę stopni w prawo.
Uwaga: wartość musi być liczbą całkowitą.

LT

Użycie: LT wartość

Obraca żółwia(ie) o podaną liczbę stopni w lewo.
Uwaga: wartość musi być liczbą całkowitą.

HT

Użycie: HT

Ukrywa aktywne żółwie.

ST

Użycie: ST

Pokazuje aktywne żółwie.

PU

Użycie: PU

Wyłącza zostawianie śladu przez żólwie.

PD

Użycie: PD

Włącza zostawianie śladu przez żólwie.

CS

Użycie: CS

Czyści planszę.

REPEAT

Użycie: REPEAT ilość powtórzeń [polecenia]

Powtarza podane w nawiasach kwadratowych polecenia podaną ilość razy. Ilość powtórzeń jest liczbą całkowitą.

TELL

Użycie: TELL ID żółwia lub [id żółwi]

Tworzy lub wybiera żółwie. ID żółwia nie może być wyższe o 2 od najwyższego istniejącego żółwia.

ASK

Użycie: ASK id żółwia lub [id żółwi] [polecenia]

Wykonuje podane w nawiasach kwadratowych polecenia dla wybranych żółwi. Może tworzyć nowe żółwie.
